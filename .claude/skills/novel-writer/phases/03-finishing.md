# フェーズ3：仕上げ（通読チェック＋最終出力）

## このフェーズの目的

フェーズ2で章ごとにレビュー済みの原稿を**全体として通読**し、章をまたいだ問題を修正した上で、最終稿として結合・出力する。

> フェーズ2の章単位レビューでは各章の品質は担保されている。このフェーズで行うのは**章単位では検出できない全体レベルの問題**の修正のみ。

---

## ステップ1：全体通読レビュー

### やること

`02_writing/` 配下の全章を結合した全文をサブエージェントに渡し、以下の観点でレビューさせる。

### 呼び出し方

```
Agent(
  subagent_type: "general-purpose",
  prompt: agents/full-review.md の内容 + 全章テキスト全文
)
```

レビュー指示の詳細は以下の独立ファイルに定義：

**`agents/full-review.md`**

### レビュー結果の扱い

- 指摘があれば該当章を修正し、修正版を `03_finishing/` に保存
- 指摘がなければ `02_writing/` の原稿がそのまま最終稿となる

---

## ステップ2：最終出力

全章を1ファイルに結合し、以下の形式で出力する。

### 出力先

```
03_finishing/final.md
```

### 出力形式

```markdown
# [作品タイトル]

---

## 第一章：[章タイトル]

[本文]

---

## 第二章：[章タイトル]

[本文]

...
```

---

## チェックリスト（フェーズ完了条件）

- [ ] 全章を通読し、章をまたいだ問題がないことを確認した
- [ ] 問題があった場合、修正を行った
- [ ] 全章を1ファイルに結合し、final.md として出力した

# clients/ — クライアント・サービス別参照フォルダ

各クライアント/サービスごとにフォルダを作成し、過去資料・ログを蓄積する。
`_input.md` の「クライアント/サービス名」欄に**このフォルダ名と一致する名前**を書くと、
pptxコマンド実行時に自動的に参照される。

## フォルダ構造

```
clients/
├── {クライアント名}/
│   ├── references/   ← 参考資料・過去パワポ・提案書など
│   └── logs/         ← 過去の議事録・メモ・やりとりログ
└── ...
```

## 追加方法

新しいクライアントを追加する場合:
```
clients/{クライアント名}/references/
clients/{クライアント名}/logs/
```
の2フォルダを作成して資料を配置する。

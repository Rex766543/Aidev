# 最後の晩餐会 ―美食家は二度死ぬ―

マーダーミステリー / 6人用（4〜8人対応）/ 90〜120分 / GM1名必要

## あらすじ

「食の皇帝」毒島鯛造が主催する究極グルメ晩餐会。孤島の私邸「美食城」で、メインディッシュ提供後に毒島が倒れ死亡。毒殺。嵐で警察は来られない。犯人はこの中にいる。

## 特徴

- **二段構えの推理**: 前半で到達する「偽の結論」が、中盤の鑑定で覆る
- **全員に見せ場がある**: 各キャラが推理に不可欠な情報を持つ
- **犯人プレイが楽しい**: 犯人には複数の防御手段がある
- **ブラックコメディ**: 料理名やレビューの笑える要素と本格推理の両立

## 遊び方

1. GMは `gm/guide.md` と `design/truth.md` を事前に読む
2. `rules/rules.md` と `handouts/` を印刷
3. `cards/` を印刷・カット（フェーズ1用7枚、フェーズ2用10枚）
4. 進行台本に沿って進行

# ゲーム設計書（改訂版）

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | **「最後の晩餐会 ―美食家は二度死ぬ―」** |
| ジャンル | ブラックコメディ × 本格ミステリー |
| プレイ人数 | 6人（基本）/ 4人最小・8人最大 |
| プレイ時間 | 90〜120分 |

## 舞台設定

美食評論家 **毒島 鯛造** が主催する究極グルメ晩餐会。孤島の私邸「美食城」。毒島は「食の皇帝」を自称する傲慢な老人。招待された6人は全員毒島と因縁がある。メインディッシュ提供後、毒島が倒れ死亡。毒殺。孤島で警察は来られない。

## ゲームの二段構え

**前半（フェーズ1〜2）**: 全証拠が金沢を指す → プレイヤーは「金沢が犯人」と確信

**転換点（フェーズ3）**: 緑川の鑑定で「ワインの中身は下剤。毒ではない」と判明 → 全てが覆る

**後半（フェーズ4〜5）**: 3人の証言（蒼井・黒田・赤羽）が揃って初めて白石が犯人と確定

## 勝利条件

- **犯人以外**: 真犯人を当てる + 秘密ミッション達成
- **犯人（白石）**: 投票で選ばれない + 秘密ミッション達成
- **偽犯人（金沢）**: 殺人犯として投票されない + 誠の名誉回復

## トーン

- 料理名が大仰（「深淵より這い出でし漆黒のデミグラス〜牛頬肉の慟哭〜」）
- 毒島のレビューが酷い（「犬の餌以下」「原始人シェフ」「バカ舌で生きるな」）
- 事件は深刻だが、随所にコメディ要素
- エンディングは本格的な悲劇（毒島の償い+金沢の皮肉な運命）

## 整合性チェック

### 偽の結論（金沢犯人）到達ルート ✅
- フェーズ1のカード(E1-01〜04) + 白石の証言 → 議論1で「金沢が怪しい」
- フェーズ2前半(E2-01, E2-03, E2-04) → 「金沢確定」の空気
- **これは設計通り。プレイヤーをこの結論に誘導する**

### どんでん返しルート ✅
- **E2-02（緑川の鑑定）が唯一のトリガー** → これが出た瞬間にゲームが変わる
- E2-02を誰かが取得し、緑川に鑑定を依頼する必要がある
- → **GMは必要に応じて「ワイングラスの沈殿物をもっと詳しく調べたい人はいますか？」とヒントを出す**

### 真犯人到達ルート ✅
- **最短**: 緑川の時間推論(#14-16) + E2-08(シャンパングラス) + 蒼井の証言(#18) + E2-05(遺言書の横領記述)
- **推奨**: 上記 + 黒田の証言(#19) + E2-09(来島記録) + E2-10(手帳)
- **3人の証言が揃う必要がある**: 蒼井(グラス目撃) + 黒田(電話目撃) + 赤羽(遺言書再読)
- → **1人でも情報を出し渋ると解けない。これが議論の駆動力になる**

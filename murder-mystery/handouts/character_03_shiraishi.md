## プロフィール

**白石 玲奈 / 35歳 / フリーライター（元・毒島の秘書）**
毒島鯛造の秘書を5年間務めた後、1年前に退職。現在はフリーライターとして活動中。冷静な観察眼の持ち主。毒島の身の回りのことを最もよく知る人物。

## あなたの物語

私は白石玲奈。毒島鯛造を殺しました。

5年間の秘書生活で、私は毒島の経費を水増しし、架空請求を繰り返し、**総額2,800万円を横領**していました。毒島のパワハラは事実ですが、辞めた本当の理由は横領の発覚を恐れたからです。

しかし退職後、毒島は会計監査で横領に気づきました。毒島は今夜の「重大発表」で、私の横領を全員の前で公表し、警察に告発するつもりでした。

私は1週間前に毒島の書斎に盗聴器を仕掛けており、この計画を知りました。告発されれば刑事罰は免れない。逃げ場はない。

だから殺しました。

### 犯行の詳細

1. 事前にトリカブトからアコニチンを抽出（秘書時代に読んだ緑川教授の著書で方法を学んだ）
2. 小さなガラス瓶に溶液を入れて持参
3. 当日15:00に「準備を手伝いたい」と他のゲストより2時間早く到着
4. 17:25、ウェルカムドリンクのシャンパングラスが給仕テーブルに並べられた際、**毒島に渡すグラスにアコニチンを滴下**
5. 17:30のウェルカムドリンクで毒島がシャンパンを飲んだ
6. アコニチンの発症は経口摂取後30分〜2時間。19:00に発症（1.5時間後）
7. 使用済みのガラス瓶は、17:35頃にテラスから海に投棄した

### あなたの表向きの設定（他のプレイヤーに語る嘘）

- **19:00** — 毒島が倒れる（計画通り）
- **19:10** — 自ら警察に通報（冷静な人物という印象を与えるため）

## 犯人としての戦略ガイド

### 序盤（フェーズ1〜議論1）: 金沢に疑惑を集中させる

- 積極的に「協力的な証人」を演じる
- 金沢の旧姓、目つき、ワインセラーへの移動を証言する
- 「私は元秘書ですから、この屋敷のことは何でも知っています。お力になれますよ」

### 中盤（フェーズ2〜議論2）: 冷静に防御する

「ワインが凶器ではない」と判明した場合、以下で防御:

| 証拠 | あなたの反論 |
|------|------------|
| シャンパングラス付近にいた | 「給仕テーブルのグラスの並びがずれていたので直しました。元秘書ですからホスピタリティは染みついています」 |
| 「今夜で終わりにする」の電話 | 「暴露本の編集者との電話です。原稿の最終締め切りが今夜だったので」 |
| 15:00に早く到着 | 「5年間通った場所です。退職してから初めてだったので、懐かしくて早めに来ました」 |
| 横領の発覚 | **「横領は認めます。でも、だからこそ毒島を殺す理由がないんです。横領犯は毒島が生きている方が都合がいい。死んだら調査が入って確実にバレます」** ← これが最強の反論 |
| 金沢をフレーミング | 「私は事実を述べただけです。疑うかどうかは皆さんの判断です」 |
| トリカブトの知識 | 「庭の植物を知っていることと、毒の抽出方法を知っていることは別です」 |

### 終盤: 他のキャラに矛先を向ける

- 「緑川教授は毒物の専門家です。鑑定結果を操作できる立場にあるのでは？」
- 「黒田さん、包丁が足りないそうですが……」
- 「蒼井さん、デザートの秘密の材料って何ですか？」

### 追い詰められた場合

全ての証拠が揃って追い詰められても、**最後まで否定してください**。「状況証拠だけです。決定的な物的証拠はないはずです」と主張する。（ガラス瓶は海に捨てたので、確かに手元にはない）

## ロールプレイのヒント

- 冷静で知的。感情を表に出さない
- 「元秘書」の立場を活かして、屋敷や毒島についての情報を持っていることを自然にアピール
- 序盤は「金沢が怪しい」と確信しているフリをする
- 中盤で自分に疑惑が向き始めたら、動揺せず論理的に反論
- 横領を認めるタイミングが重要。先に認めてしまえば「正直な人」に見える。追い詰められてから認めると「まだ隠していることがある」と思われる
- **最大のポイント**: あなたは「この中で最も冷静で知的な人物」という印象を維持すること。その印象が崩れた時が最も危険

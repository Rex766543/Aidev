# 戦闘と死 — SAOの戦闘仕様

## HP（ヒットポイント）

- プレイヤー・モンスター双方が持つ生命値
- ダメージを受けると減少し、ゼロになると死亡（消滅）
- HPバーは自分のものは常時表示。パーティメンバーのHPも視認可能
- 他プレイヤーのHPは通常見えない

### HP低下時の演出
- HPが一定割合以下になると視界が暗転し始める
- 思考が鈍り、身体の動きが重くなる感覚が生じる
- HPバーの色が変化する（緑→黄→赤）

## ペインアブソーバー

- 痛覚を減衰させるシステム機能
- 完全に痛みを消すわけではない——鈍い圧迫感や衝撃感は残る
- 設定上、痛みは本来の感覚の約10%程度に軽減されている
- ただし恐怖・精神的苦痛はシステムで軽減されない

## 死亡

- HPがゼロになるとアバターがガラス片のように砕け散り、消滅する
- デスゲーム下では、この瞬間にナーヴギアが現実の脳を破壊する（＝現実の死）
- 死因は問わない：モンスター、プレイヤー、落下、罠、毒、いずれも同様
- **復活手段は存在しない**

## モンスターの攻撃分類

モンスターのプレイヤーに対する攻撃は大きく二つの系統に大別される：
- **直接的攻撃**：HPを減らす攻撃。言うまでもなく致命的
- **間接的攻撃（デバフ攻撃）**：直接HPは減らないが、スタン・麻痺等の状態異常を付与し、間接的に大きな危険を招く
- 初心者にとって間接的攻撃のほうが対処が難しく、パニックに陥りやすい

## 戦闘の基本

### 通常攻撃
- プレイヤー自身の操作で武器を振る
- ダメージは低いが、硬直がなく連続行動が可能

### ソードスキル（→ 01_game_system.md 参照）
- システムアシストによる高速・高威力の技
- 発動後に硬直（ディレイ）が発生する
- **ブーステッドアシスト**：ソードスキル特有の重量感。システムアシストによる威力ブーストで、プレイヤーの身体能力を超えた破壊力が武器に乗る

### 戦闘テクニック

- **スイッチ**：パーティ戦闘で、一人がソードスキル発動後の硬直中にもう一人が前に出て攻撃を引き継ぐ連携技術。SAO戦闘の基本戦術
- **カウンター・パリィ**：敵の攻撃を弾き返すタイミング技
- **ノックバック**：攻撃によって相手を後方に弾き飛ばす効果
- **タンブル（転倒）**：強い衝撃で相手を転倒させる効果
- **オーバーキル**：HPをゼロにした後もさらにダメージを与えること

### 戦闘用語

- **アグロ（Aggro）**：モンスターの敵対値。攻撃やスキル使用で蓄積し、最もアグロが高いプレイヤーがモンスターの標的になる
- **アグロレンジ**：モンスターがプレイヤーを感知・攻撃対象にする距離範囲
- **ヘイト（憎悪値）**：アグロと同義。モンスターがどのプレイヤーを狙うかの優先度指標
- **DPS（Damage Per Second）**：秒間ダメージ量。火力の指標
- **ラストアタック（LA）**：モンスターにとどめを刺した一撃
- **ラストアタックボーナス**：LAを達成したプレイヤーに追加報酬（レアドロップなど）が付与される仕組み
- **MPK（モンスターPK）**：モンスターを利用して他プレイヤーを間接的に殺す行為。システム上はPK扱いにならないが、倫理的には殺人
- **定点狩り**：同じ場所に留まって繰り返しモンスターを狩る効率的なレベリング手法

### 戦闘ロール（役割）

| ロール | 英名 | 機能 |
|--------|------|------|
| タンク | Tank | 盾役。敵のヘイトを集めて攻撃を受け止める |
| アタッカー/ダメージディーラー | Attacker/DD | 主火力。ソードスキルで敵を削る |
| 前衛（フォワード） | Forward | 前線で敵と直接交戦する |
| メイサー | Macer | 棍棒使い。スタン付与やノックバックに優れる |
| フェンサー | Fencer | 細剣使い。手数と突き技に長ける |
| ナイト（騎士） | Knight | 盾と片手剣を持つ防御寄りの前衛 |

### パーティ・レイド戦闘
- 最大6人でパーティを編成可能
- パーティメンバー間ではHP・位置情報の共有が可能
- 役割分担（タンク・アタッカー・前衛）が自然発生する
- **レイド**：複数パーティの連合。フロアボス攻略の基本編成
- **フルレイド**：パーティ上限数（7パーティ＝最大48人）のレイド編成
- **野良パーティー**：ギルドに拠らず、その場で臨時に組んだパーティ
- **レイド内の部隊編成**（ボス戦時）：
  - 各パーティにアルファベットの隊名が付く（A隊〜G隊等）
  - **前進部隊（H隊等）**：ボスに最初に接敵する前衛
  - **後退部隊（E隊等）**：ローテーション待機。前衛の硬直・HP低下時にスイッチで交代
  - **サポート/後衛**：回復・バフ支援
  - レイドリーダーが「E隊、前進！」「H隊、後退！」等の声で交代を指示
  - 熟練レイドでは六段階のローテーションを組み、前線を維持し続ける

### 状態異常

| 状態異常 | 効果 |
|---------|------|
| 毒（ポイズン） | 一定時間HPが継続減少する。毒性（トキシティ）の段階がある |
| 麻痺（パラライズ） | 一定時間行動不能になる |
| スタン | 短時間の行動不能。打撃や特定スキルで発生 |
| バーサーク | 理性を失い暴走する。味方の識別ができなくなる |
| タンブル（転倒） | 倒れた状態。起き上がるまで無防備 |
| 部位欠損 | 手足などの部位が一時的に使用不能になる |

- **バフ**：一時的な能力上昇効果（料理やアイテムで付与）
- **デバフ（阻害状態）**：一時的な能力低下効果
- **キュアラブル・ペイン**：システム上「治療可能な痛み」として分類される状態

## カーソル（カラーカーソル）

プレイヤーの頭上に表示される色付きカーソルで、そのプレイヤーの状態を示す。

| カーソル色 | 状態 | 通称 |
|-----------|------|------|
| 緑（グリーン） | 通常プレイヤー（一般） | — |
| 黄（イエロー） | 犯罪行為の軽微な違反者（窃盗・圏内暴行など） | イエロープレイヤー |
| 橙（オレンジ） | 重犯罪者（殺人に至らない重大な犯罪行為） | オレンジプレイヤー |
| 赤（レッド） | 他プレイヤーを殺害した者（PK実行者） | レッドプレイヤー |

- カーソル色はシステムが自動判定する（アンチクリミナルコード/犯罪防止コードに基づく）
- 赤カーソル（レッドプレイヤー）は殺人の記録が一定期間消えない
- 隠蔽スキルやアイテムでカーソル表示を隠す手段が存在する
- **カーソルの視認**：暗所ではカーソルの色が特に目立つ。ダークプレイヤー（オレンジ/レッド）が闇の中に潜んでいると、カーソル色だけが視界に赤く染み込むように見える
- NPCにもカーソルが表示される——NPCのカーソルは黄色系で、mobモンスターのカーソルは赤系（敵性表示）
- **オレンジギルド**：メンバーにオレンジプレイヤーを含むギルド
- **レッドギルド/殺人ギルド**：PKを組織的に行うギルド

### PK（プレイヤーキル）の分類

- **PK**：プレイヤーを直接殺害する行為
- **MPK（モンスターPK）**：モンスターを誘導して他プレイヤーにぶつけ、間接的に殺す行為
- **圏内PK/圏内殺人**：本来不可能な圏内での殺人。システムの裏をかく特殊な手段で実現される
- **殺人プレイヤー**：PK行為を行う犯罪者プレイヤーの総称

## 圏内・圏外の戦闘ルール

| | 圏内（セーフゾーン） | 圏外（フィールド・ダンジョン） |
|---|---|---|
| PvPダメージ | 無効 | 有効 |
| モンスター出現 | なし | あり |
| 詐欺・窃盗 | 可能 | 可能 |
| 殺人 | 不可能（※デュエル全損決着を除く） | 可能 |

## 武器破壊（ウェポン・ディストラクション）

- 戦闘中に武器の耐久度がゼロになると武器が破壊される
- 片手用直剣の場合、刀身に横方向から亀裂が入り、折れるように砕ける
- 破壊の兆候：耐久値が危険域に入ると「ぴきっ」という罅割れ音が聞こえる
- 武器を失ったプレイヤーは素手戦闘（格闘スキル）か逃走を強いられる
- ボス戦中の武器破壊は致命的——装備変更のための数秒の隙が命取りになる

## 全層共通ルール

- アインクラッドは100層で構成される巨大な浮遊城（鋼鉄の円錐形）
- **下層ほど広く、上層ほど狭い**。第一層が最も広大（直径約10km）
- 各層に最低1つの主街区（セーフゾーン）が存在し、転移門・宿屋・NPC店舗を備える
- 気候は層ごとに固定（雪の層、砂漠の層、常夏の層など）
- 昼夜サイクルは現実と同じ24時間周期
- 上層ほど物価が高く、モンスターが強い
- 仮想世界の夕焼け・星空は「言葉を失うほど美しい」が、よく見ると人工性がある

## 層構成の基本要素

各層は以下のパーツで構成される：

| 要素 | 説明 |
|---|---|
| 主街区 | 転移門広場を中心としたセーフゾーン。NPC店舗・宿屋・施設あり |
| フィールド | 街の外の冒険エリア。モンスター出現。PvP有効 |
| 迷宮区 | 各層のメインダンジョン（タワー型）。最深部にフロアボスの部屋 |
| フィールドダンジョン | フィールド内に点在する小規模ダンジョン |
| 安全地帯（迷宮区内） | ダンジョン内の小規模休憩スポット。モンスター非出現 |

## 層間移動手段

- **螺旋階段**：フロアボス撃破後に出現。次の層へ物理的に繋がる
- **転移門**：主街区間を瞬時移動。攻略組が到達・開放することで利用可能に
- **転移結晶**：アイテム使用で転移門のある任意の街に瞬時移動

# キャラクター設計

## ソウヤ / 偽名「カイト」（主人公）

| 項目 | 内容 |
|------|------|
| 名前 | ソウヤ（本名）/ カイト（偽装名） |
| 立場 | 中層ソロプレイヤー。元オレンジギルド《灰狼団》の末端メンバー |
| 目的 | 過去を隠して静かに生き延びること。だが心のどこかで「隠れ続ける自分」に耐えられなくなっている |
| 性格の核 | 臆病な善人。逃げることで正しさを守った人間 |
| 認知の偏り | 「ゲームだ」寄りで過ごそうとするが、仲間の血を見るたびに「現実だ」に引き戻される |
| 新規設定との関わり | 《偽装》スキルの使い手。このスキルで「カイト」として生きている |

### 背景
- 《灰狼団》は中規模のオレンジギルドで、圏外での強盗・MPKを主な活動としていた
- ソウヤは加入当初、灰狼団がオレンジギルドだとは知らなかった（勧誘時は一般ギルドを装っていた）
- ギルドの本性に気づいたとき、すでに行動を共にした事実が「共犯」として機能する状況に追い込まれていた
- **灰狼団内でのソウヤの役割は「斥候」だった**。隠蔽スキルの高さを買われ、ターゲットの動向を探る偵察任務を担当。PKの実行現場には参加させてもらえなかった——末端メンバーに犯罪の直接証拠を持たせないのが灰狼団の組織運営方針だった（離反時にカーソル色で足がつかないよう、実行犯と偵察を厳密に分離していた）
- ある夜、偵察として報告した情報がもとで一人の中層プレイヤーが殺害されることを知り、実行前に現場から逃げ出した。**システム上、ソウヤは一度も他プレイヤーに攻撃を加えておらず、カーソルはグリーンのまま保たれている**
- 逃走後、隠蔽スキルの極限的な酷使の中で解放された《偽装》スキルで名前とカーソルを書き換え、「カイト」として第三十五層に逃げ込んだ
- 灰狼団はその後、攻略組の有志による討伐で壊滅したが、被害者遺族（ダイチ）はメンバーの追跡を続けている
- **カーソルがグリーンであることは、ソウヤの「手は汚れていない」証明であると同時に、「自分の情報で人が死んだ」という間接的加担の罪を免罪しない**。システムが裁かない罪を、ソウヤは自分で背負い続けている

### 現実世界の記憶
- 現実では高校一年生だった。ゲーマー仲間に誘われてSAOを購入した
- 両親と妹がいるはずだが、顔がぼやけ始めている。妹の声だけはまだ覚えている
- 「現実に帰っても灰狼団での日々を知る者はいない——逃げた過去は仮想世界だけの記憶になる」。それが救いなのか新たな隠蔽なのか、ソウヤ自身にもわからない
- この「帰ったら全部なかったことになる」という認知が、偽装を続ける言い訳の一つになっている

### 戦闘スタイル
- 片手剣使い。偽装中はソードスキル使用不可のため、通常攻撃と体術スキルのみで戦う
- 隠蔽スキルの熟練度が極めて高い。先制攻撃と逃走が得意
- 偽装解除時は片手剣のソードスキル（ホリゾンタル、バーチカル・スクエア等）を使える中級レベルの剣士

## ミナ（鍛冶師）

| 項目 | 内容 |
|------|------|
| 名前 | ミナ |
| 立場 | 攻略組の後方支援鍛冶師。前線に出ることもある稀有な生産プレイヤー |
| 目的 | 第三十五層迷宮区の特殊鉱石を入手し、攻略組用の武器を打つこと |
| 性格の核 | 実利的な楽天家。「使えるものは使う」主義だが、人を見捨てられない |
| 認知の偏り | ゲームと現実の境界に無頓着。「どっちでもいい、今ここで生きてるのは本当でしょ」 |
| 新規設定との関わり | カイトの武器の異常な消耗に最初に気づく。クライマックスで耐久度回復という形で物語を決定づける |

### 特徴
- 茶色の革エプロンと携帯炉が常備装備。ブラックスミス・ハンマーを腰に下げている
- 武器の状態を見るだけでおおよその耐久度を推定できる腕前
- 攻略組の前線に同行した経験があり、戦場の空気に慣れている
- 人懐っこいが観察力が鋭い。カイトの不自然さに気づくが、すぐには追及しない

### 内面の葛藤
- かつて自分が打った武器を持った攻略組メンバーがボス戦で死んだ経験がある。武器の耐久度が足りなかった——少なくともミナはそう思っている
- 「自分は安全な場所で金槌を振るっているだけ。前線で死ぬのは、自分の武器を握った誰か」という後ろめたさを抱えている
- だからこそ前線に同行する。自分の武器が折れる前に、自分の手で直す。それがミナにとっての「戦い方」
- クライマックスでカイトの武器を応急修理する行為は、単なる鍛冶師の機能ではなく「今度こそ自分の手で、この武器を持つ人間を生き延びさせる」というミナ自身の選択になる

## ダイチ（復讐者）

| 項目 | 内容 |
|------|------|
| 名前 | ダイチ |
| 立場 | 中層プレイヤー。かつてのパーティメンバーが《灰狼団》に殺され、犯人を追っている |
| 目的 | ソウヤを見つけ出し、仲間の死の真相を突き止めること。復讐か正義かは自分でもわかっていない |
| 性格の核 | 執念深いが公正。事実を曲げてまで人を裁かない |
| 認知の偏り | 「現実だ」寄り。ゲーム用語でPKを語る風潮を嫌う |
| 新規設定との関わり | 偽装スキルの存在を知らないが、「カーソルがグリーンでも信用できない奴はいる」と直感している。最終的にソウヤのグリーンカーソルを見て真実を受け入れる |

### 特徴
- 両手剣使い。戦闘力は中層上位
- 情報屋から灰狼団メンバーの情報を買い集めている
- 感情を抑制しようとするが、仲間の話になると声が震える
- 「正義」と「復讐」の境界で揺れ続けている。ソウヤのグリーンカーソルがその揺れに終止符を打つ

## ヨシムラ（パーティ前衛）

| 項目 | 内容 |
|------|------|
| 名前 | ヨシムラ |
| 立場 | ミナのパーティの前衛。片手剣＋盾のナイト型 |
| 目的 | ミナの護衛と迷宮区攻略 |
| 性格の核 | 生真面目な常識人。疑い深いが、疑う自分を嫌っている |
| 認知の偏り | 「ゲームだ」と「現実だ」の間で最も揺れるタイプ。合理的判断と感情のぶつかり |
| 新規設定との関わり | カイトがソードスキルを使わないことへの不審を最初に口にする役割。重傷を負うことで、カイトに「偽装を解く覚悟」を迫る触媒 |

### 特徴
- パーティのリスク管理を担う常識的な前衛。盾防御（ストロングウォール）が得意
- 「スイッチもできない奴と迷宮区に入るのは正気じゃない」と正論を言うが、ミナに押し切られる
- カイトに対する不信は個人的な悪意ではなく、パーティの安全を考えた合理的判断
- 重傷を負った後も「なぜ使わなかった」と問い続ける。答えを聞く前に気を失う

## サエ（パーティ後衛・回復役）

| 項目 | 内容 |
|------|------|
| 名前 | サエ |
| 立場 | ミナのパーティメンバー。槍使い兼回復担当 |
| 目的 | パーティの生存率を上げること。「全員で帰る」が口癖 |
| 性格の核 | 穏やかで面倒見がいい。だが怒ると一番怖い |
| 認知の偏り | 「現実だ」寄り。ポーションを「薬」と呼び、HPを「命」と言い換える |
| 新規設定との関わり | 直接は関与しないが、第4章でヨシムラの応急処置を担い、パーティが壊滅しないための最後の砦として機能 |

### 特徴
- 槍のリーチを活かした中距離戦闘と、ポーション投与のタイミング管理が役割
- ミナとは古い付き合い。ミナが前線に出ると言い出したとき、最初に「行く」と答えた
- 戦闘中は冷静だが、仲間のHPバーが黄色に変わると声が硬くなる

## トウジ（パーティ斥候）

| 項目 | 内容 |
|------|------|
| 名前 | トウジ |
| 立場 | ミナのパーティメンバー。短剣使い。索敵・罠解除担当 |
| 目的 | 迷宮区のマッピングと安全確保 |
| 性格の核 | 寡黙で実務的。必要なことしか言わない |
| 認知の偏り | 「ゲームだ」寄り。効率を最重視し、感情を表に出さない |
| 新規設定との関わり | 霧の中での索敵を担当するが、常霧の層では索敵スキルが減衰するため、自身のスキルの限界に苛立つ。カイトの隠蔽スキルの高さには一目置いている |

### 特徴
- パーティの「目」。迷宮区では先行して罠と敵配置を確認する
- カイトに対しては中立。ソードスキルを使わないことより、隠蔽と索敵の腕前を評価する合理主義者
- 第4章のネームドMob奇襲は、霧による索敵減衰でトウジの警戒網をすり抜けたことで発生——トウジにとっても自分の限界を突きつけられる場面

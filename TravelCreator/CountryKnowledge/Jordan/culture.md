# Jordan — 文化・習慣・社会

## 国民性・社会構造

### 一言で
イスラムを土台にしつつ、王制・家族・共同体を重視する多層社会。

### 社会の拠り所
- 国教：イスラム（スンニ派が中心）
- 制度：宗教だけでなく、家族・親族・地域共同体・王制への安定期待が大きい
- 国民性（粗くまとめると）：もてなし・礼儀・家族重視・対立回避・穏健さ・現実感覚
- 「西欧的個人主義」より「家族・共同体・礼節・安定」の比重が高い
- 注意：アンマンの若者と地方の保守的共同体では感覚がかなり違う

## ヨルダン系（East Bankers）とパレスチナ系の違い

### 前提
完全に二分できるものではない。世代・階層・都市か地方か・家族の歴史でかなり違う。

### East Bankers（ヨルダン系・東岸系）
- 歴史的に部族・地方・王制・軍や治安機構とのつながりが比較的強い
- 国家の土台として位置づけられてきた
- 体制忠誠の主要基盤（Brookings）

### パレスチナ系ヨルダン人
- 元々パレスチナ地域出身の人・その子孫（現在はヨルダン国民）
- 都市商業・専門職・民間経済・都市中間層の比重が比較的大きい
- パレスチナ問題への感度が高い

### 街中での現れ方
- アンマン西側の商業的・都市的エリア：歴史的にパレスチナ系住民の存在感が強いと言われる
- 地方都市・部族ネットワーク強い地域・国家機関：東岸系アイデンティティが見えやすい
- 日常生活では**かなり混ざっており**、外見で判別は不可能
- 違いが見えるのは：話題、家族の出自、働く分野、政治への距離感

### 現在の関係
- 日常社会：かなり混ざっている（同じ学校・職場・商店街、婚姻・交友あり）
- 政治・代表性・国家アイデンティティになると線が見えやすくなる
- 選挙制度や国家機関の採用では東岸系優位・パレスチナ系の過小代表が論点

## ヨルダンとイスラエル・パレスチナ問題

### ヨルダン国家の立場
- イスラエルと国交あり（和平条約）
- 国家としては安定・外交を優先
- ヨルダン国内 ≠ イスラエルと直接対立している状態ではない

### 国民感情
- パレスチナ系住民が多く、パレスチナ問題への共感は大きい
- 政府（現実的）と国民感情（共感的）は分かれている

### 「ユダヤ vs アラブの対立」という理解について
- 正しい部分：歴史的・政治的には対立あり（ガザ・西岸問題など）
- 違う部分：ユダヤ人 ≠ 全員イスラエル政府、アラブ人 ≠ 全員対立
- 本質：民族対立というより**政治・領土問題**

## アンマンの景観 — 「途上国的に見える」理由の構造的理解
*Related: Amman*

### 「同じ店ばかり」に見える理由
- 小規模商店が同じ卸業者・同じ輸入ルートを使う → 品揃えが自然と収束
- リスク回避のため「売れる定番商品」に特化 → 多様性が出にくい
- インフォーマル経済（無認可・零細）の比率が高く、参入コストが低い → 似た業態が乱立

### 「雑然として見える」理由
- 都市計画・行政の整備ペースより人口増加・経済発展が先行した
- 未完成の建物（tax avoidance）、露店、不統一な看板は「遅れている」のではなく、**制度・行政・経済構造の違い**が現れたもの

### 野良猫・犬が多い理由
| 要因 | 内容 |
|------|------|
| 都市管理の不足 | 去勢・避妊プログラムが十分ではない |
| ゴミ処理 | 食べ残しが道路に多く、食料源として猫・犬を引き寄せる |
| イスラームの動物観 | 虐待・殺処分を禁止する思想（ハディースに「猫を殺してはならない」の伝承あり）→ 行政が殺処分しにくい文化的背景 |
| 複合的要因 | 「イスラームだから」だけでなく、管理インフラ・ゴミ・文化の組み合わせ |

## 乗合ミニバス（セルビス / Minibus）
*Related: Amman*

### 「無秩序に見えて実は認可されている」構造
- **完全無許可ではない**：路線・車両・営業許可は行政から付与されている（半公式運用）
- **個人オーナー制**：日本の大手バス会社のような組織ではなく、小規模個人事業者が認可を取って車両を運行
- 公共交通（正式なバス・地下鉄）が未整備な地域の生活インフラとして機能
- 需要が多い路線に車が集まり、有機的にルートが形成されてきた歴史がある

### 「なぜあんなに大量に成立するか」
- 需要が多い（都市の地形・分散したエリア）
- 参入コストが低い（車+認可があれば個人で開業できる）
- 公共交通の代替として市民に根付いた生活インフラ

## 中東建築：なぜ石窟（岩削り）なのか

### 結論
環境 × 防御 × 技術の**最適解**

### 理由
1. **気候**：砂漠は日中激暑・夜寒い → 岩の中は温度安定（エアコン不要）
2. **素材**：石は豊富、木材は少ない → 「積む」より「削る」
3. **防御**：外敵から見えにくい、侵入しにくい → 要塞として優秀
4. **水管理**：岩に水路・貯水を組み込める → ナバテアの強みと直結

→ 自然環境に最適化したインフラ

## 商人文化の比較（歴史的）

各商人が強い理由は「信頼とネットワークをどう作るか」の違い。

| 商人 | 強さの源 | 本質 |
|------|---------|------|
| ナバテア人 | 環境適応・ルート支配・水管理 | 砂漠を物流インフラ化 |
| イスラーム商人 | 宗教ベースの信用経済（利子禁止→共同事業、ザカート→社会安定） | 信用のインフラを宗教で作った |
| ユダヤ商人 | ディアスポラ（散在）ネットワーク・高い識字率・仲間内信用 | グローバルネットワーク最強タイプ |
| カルヴァン派 | 労働＝神の使命、勤勉・倹約・再投資 | 資本主義を回す思想エンジン |

### ナバテア商人の詳細
- 遊牧・半遊牧的背景（ベドウィン的属性）を持ちながら交易の主体でもあった
- 単なる「運送業者」でも「転売屋」でもなく、**物流＋商社＋インフラ運営を全部やった**
- 参入障壁：砂漠の水・ルート知識を独占（外部商人は単独では不可能）
- 中立ポジション：特定大国に依存せず、どことも取引（敵を作らない）

## ワディラムのベドウィン部族

### 主要部族
- **ザラビア族（Zalabia）**：ワディラム一帯の主要部族
- **フウェイタート族（Huweitat）**：アラビアのロレンスと共にオスマン帝国に抵抗、アカバ攻略に協力 — 部族の誇り

### 族譜（نسب）の重要性
- ベドウィンのアイデンティティの核：部族（قبيلة）→氏族（عشيرة）→家系の順に語れることが誇り
- 「أنت من أيّ قبيلة؟」（どの部族ですか？）を聞くと年配者は喜ぶ

### 見分けのヒント（商業ベドウィン vs 深い繋がりを持つベドウィン）
- 族譜をすらすら語れる → 本物のベドウィン文化の担い手
- 実用的なラクダ・ヤギを所有（観光ライドだけでない） → 遊牧との繋がり残存
- 砂漠知識（星・水場・植物）を具体的な地名・経験談で語れる → 本物の確率高
- ワディラム村内の年配男性 → 商業化されていない層に近い
- 現在の大半は「定住しているがベドウィンである」ポスト遊牧状態 — 生活形態ではなく価値観・部族帰属がアイデンティティ

# Jordan — 歴史・遺跡・王朝

## ナバテア王国

### 年表（312 BC – AD 500）

| 時期 | 出来事 |
|------|--------|
| 前6世紀 | アラビア半島北部からヨルダン南部へ移動開始 |
| **312 BC** | 歴史上の初記録：アレクサンドロス大王の将軍アンティゴノスの攻撃を撃退（at-Tafila近くの岩山要塞「セラ」にて）。歴史家ディオドロスによれば、ナバテア人男性は交易見本市に出かけていたため攻撃が失敗 → **初期から交易が生活の中心** |
| 前3世紀 | プトレマイオス朝とセレウコス朝の政治的空白を利用してハウラン・ナカブ・シナイへ拡大 |
| **168 BC** | アレタス1世を初代王として王国を建国。首都をラクム（Raqmu = ペトラ）に定める |
| 前2世紀末 | 遊牧から定住化が進む。水工学インフラ整備、独自のナバテア芸術・建築・文字が生まれる |
| **84-60 BC** | アレタス3世「フィルヘレン（ギリシャ愛好者）」。ダマスカスまで版図拡大。ギリシャ・ローマ的近代化を推進 |
| **9 BC – AD 40** | **アレタス4世「フィロデモス（民の愛好者）」**。ナバテア文明の絶頂期 |
| AD 75-106 | ラベル2世（最後の王）の時代。ローマが紅海ルートに貿易路を変えたことで陸上キャラバン支配が弱体化。首都をハウランのボスラに移転 |
| **AD 106** | トラヤヌス帝がナバテア王国を併合 → ローマ属州「アラビア・プロウィンキア」に。文化はローマ支配下でも継続。首都は「ペトラ・メトロポリス」→ 後に「パレスティナ・テルティア」 |
| 4世紀 | キリスト教に改宗 → ペトラが司教座になる |
| 630年代 | イスラム征服時の記録にペトラはほとんど登場しない（すでに衰退していた） |

### 一言で
砂漠を通るビジネスを全部握っていたプレイヤー。「砂漠版GAFA」的存在。

### なぜ歴史的に重要か
- 東西交易（香料・絹）を支配
- ローマに匹敵する経済力を持った
- 砂漠で都市を成立させた

### 何をしていたか（3レイヤー）
| レイヤー | 内容 | 現代の例え |
|---------|------|-----------|
| 運送 | キャラバンで商品を運ぶ | ヤマト運輸 |
| 売買 | 南アラビアで香料を安く買い、北（地中海側）で高く売る | 商社・転売 |
| インフラ支配 | 水供給・ルート独占・安全確保 | 高速道路会社＋警備 |

→ この3つを**全部**やっていたのがナバテアの強み

### なぜ砂漠で都市が成立したか
- 立地：紅海とアカバ方面、シリア・地中海、アラビア内陸を結ぶ交易結節点
- 水工学：周辺の泉（アイン・ムーサ等）＋雨季の洪水水を徹底回収・制御
  - diversion dam（洪水制御ダム）
  - Muthlim tunnel（トンネル）
  - 導水路（aqueducts）
  - 貯水槽・地下シスタン
- 戦略：遊牧的機動力を持ちながら、定点インフラを構築して交易を独占

### 盗賊との関係（重要）
略奪より「継続課金」の方が儲かる論理で、盗賊をエコシステムに組み込んだ。
- 通行料を支払う（関所的な非公式税）
- 部族を護衛として契約し「警備会社化」
- ルート・水を握ることで他勢力を従属させる

→ 盗賊 vs 商人ではなく、**同じエコシステム内の役割分担**

### 文化的特徴（重要3つ）
1. **水管理**：雨水の貯水・導水 → 砂漠で都市を成立
2. **岩削り建築**：建築ではなく"削る"（ペトラ）
3. **多文化融合**：ギリシャ＋中東＋ローマ → 国際交易都市として顧客適応

### なぜ多文化を受け入れたか
→ **ビジネスのため**（戦略的合理主義）
- 交易の中心 = 文化の交差点
- 来る商人に合わせた「分かりやすい文化」への適応
- 実利重視・排他的でない宗教観
- 現代でいう「グローバル企業のローカライズ」に相当

### なぜ世界史の教科書に出ないか
- 大帝国・制度・普遍史を主軸にする叙述では主役化しにくい
- フェニキア人のほうがアルファベット・地中海史と結びつけやすく扱いやすい
- 106年にローマに併合され「属州アラビア」として歴史に吸収されて見えにくくなった
- 重要性の問題ではなく、物語の作り方の問題

### ローマへの併合（106年）
- ローマ帝国に属州アラビアとして組み込まれる
- その後はローマ史・ビザンツ史に吸収

## ペトラ（ナバテア王国の首都）

### 位置づけ
世界遺産。中東最重要遺跡の一つ。「ただの遺跡」ではなく「水制御で成立した商業インフラ都市」として見ると理解が深まる。

### 建築の役割分け

| カテゴリ | 代表例 | 役割 |
|---------|--------|------|
| 墓・霊廟（岩削り） | エル・ハズネ、王家の墓群 | 王・貴族の葬礼儀式。権力の永続性を岩に刻む |
| 神殿 | カスル・エル・ビント | 主神ドゥシャラ・アル＝ウッザーへの礼拝・宗教儀式 |
| トリクリニウム（宴会場） | アド・デイル（後世に「修道院」と命名）| 死者を偲ぶ宴会・商取引の締結の場 |
| 水・商業インフラ | ダム・水路・貯水槽・列柱道路 | 都市の生命線。これなしに交易都市は成立しない |

**「見せる」と「使う」の二層構造**
- ファサード（正面）：豪華なギリシャ・ローマ・エジプト混合様式で壁面に彫る
- 実際の墓室：シンプルで小さい
- → 「見せること自体が機能」。権力・富・洗練さを来訪商人に示すパフォーマンス建築

**建築様式の多文化融合＝ビジネス戦略**
- 地中海商人にはギリシャ様式、エジプト商人にはエジプト様式が見える設計
- → 「取引相手の文明の言語で語りかける」ローカライズ戦略が建築レベルで実装されている

**岩削りにした理由**
- 木材が砂漠にない → 積み上げ建築の素材不在
- 岩は無限にある → 素材コストゼロ・耐久性高い
- 唯一の例外：カスル・エル・ビント（神殿）だけが石積み → 礼拝者が内部を使う空間が必要だったため

### 主要スポット
| スポット | 意義 |
|---------|------|
| エル・ハズネ（宝物殿） | ペトラの象徴。岩を彫って作った高度技術 |
| アド・デイル（修道院） | 最大級建造物。交易都市のスケール感が分かる |
| カスル・エル・ビント | ナバテア唯一の石積み神殿。主神ドゥシャラの神殿 |
| ペトラ遺跡全体 | ナバテア王国の首都 |

## ナバテアの陶器（Nabataean Pottery）

### 一言で
世界最薄クラスの「卵の殻陶器（eggshell ware）」を生産した。機能性ゼロ・美と儀式のための器。

### 生産体制
- 主要生産拠点：ペトラ近郊の**アズ・ズッラバ工房**（考古学サイト入口付近）
- 粘土の産地：アイン・アット・ティーナハ
- **驚くべき事実：王国全土で大量に出土するが、国境の外ではほとんど出土しない**
  → 輸出品ではなく、内需・文化的アイデンティティの産物
- **主要工房はおそらく1つだけ**、製法を守った**一家族の職人が世代を超えて継承**

### 2種類の陶器
| 種類 | 用途 | 特徴 |
|------|------|------|
| 粗陶器（Course ware） | 調理・輸送などの実用 | 周辺地域の陶器と大差なし |
| **薄手精陶器（Fine/eggshell ware）** | 儀式・宗教用途 | 世界最薄クラス。神殿・墓で出土 |

### 薄手精陶器が薄い理由
- 粘土の可塑性が極めて高い → 薄く成形可能
- 粘土に鉄分が多い → 典型的な赤色に焼き上がる
- すべてろくろ成形（wheel-made）

### 文様の変遷
| 時期 | 特徴 |
|------|------|
| 前1世紀前半（初期） | 薄赤地に赤い波線模様。やや厚め |
| 前1世紀後半 | 線・点が自然主義的な葉模様へ進化 |
| 後1世紀後半 | 金属的な硬い質感。動物（鳥が葡萄を食べる）が登場 |
| 後2世紀末以降 | 品質低下。壁が厚くなり模様が単調に。大量生産化 |

### 開口部の広い碗（Open bowl）
- 曲底＋内側に絵付け → 実用には不向き → **儀式・宗教用**と判断
- 神殿・墓で多く発見（住居では少ない）

## ナバテアの建築（詳細）

### 建設を担った職人たち
- 碑文にナバテア人の名前で記録された職種：**建築家・彫刻家・石工・測量士・画家**
- 公共建築は必ず**王命（勅令）によって建設**（献堂碑文に明示）

### 2種類の建設工法
1. **岩削り（Rock-cut）**：自然の岩壁を上から下に向かって彫る
   - 墓・一部の住居に使用
   - 未完成の墓の観察から「上から下へ」「事前計画に基づく定式」で施工することが判明
   - 各種サイズの金属ノミを使用
2. **積み上げ（Free-standing / Built）**：地域の素材を積む
   - 素材は地域によって異なる：泥レンガ（アカバ）、玄武岩（ハウラン）、石灰岩（シャラー）、砂岩（ペトラ）
   - 岩削り墓の内部から切り出した石も建材として再利用

### 床・天井の仕上げ
- 床：石・大理石・モザイク・オプス・セクティレ（石片象嵌）
- 天井・屋根：柱・アーチ・迫持（corbelling）
  - 迫持はハウラン地方で特に多用（硬い玄武岩は豊富だが大木が少ないため）

### 建設マネジメントシステム
- 石に**作業ガイドラインの印**をつけて職人を指示
- **アルファベット・数字による記号システム**：土台と柱の関係、各部材の配置位置を示す
- ペトラの「翼ある獅子の神殿」近くの大理石工房で**1,000点以上の製作途中・完成部材**が発見

### 装飾
- 最も一般的：**型押し漆喰（Moulded stucco）**
- 壁画：住居・宗教建築の両方に使用。アレクサンドリア・ポンペイのモチーフをナバテア独自のタッチで応用

## ベドウィンと砂漠社会

### ベドウィンとは
「民族名」ではなく**生活スタイル（遊牧・砂漠適応）の属性名**。アラビア語の「Badū」に由来。
- ラクダで移動
- 水源知識を持つ
- 部族単位で動く
- 名誉・客人保護・部族連帯が重要規範

### 現在のヨルダンでのベドウィン
- 完全遊牧の人は少なく、多くは定住化
- 人口の1割未満（Britannica）
- 国家形成・軍の文脈で重要な位置を占めてきた

### ナバテアとベドウィンのつながり
ナバテア人も初期には遊牧・半遊牧的背景を持つとされる。
→「移動する砂漠民が、交易の必要から定点インフラを持ち始めた」という連続性がある

### 客人保護（ホスピタリティ文化）とは
砂漠社会の根本的な生存ルール。
- 来た人を守り、もてなす義務（敵でも数日間は無条件保護）
- 食事・水を提供する
- 背景：砂漠は水・食料がなく、単独では死ぬ → 相互扶助が生存条件
- 機能：信用インフラ（ホテル＋警備の役割）
- 中東・砂漠文化全体に共通するルール（ベドウィンに特に強い）

### ベドウィンと聖カトリーナ修道院（シナイ）
ベドウィンが修道院を襲撃した理由：
1. 生存（食料・水・物資確保）
2. 砂漠の孤立した「資源拠点」として価値がある
3. 国家統治が弱い空白地帯の論理（力ある側が取る）
4. 襲撃 → 保護契約への交渉手段

実態：多くの時代で、ベドウィンは修道院を**保護**もしていた（食料・ガイド・警備提供）。
→ **敵でもあり、パートナーでもある**関係

砂漠社会の本質：「奪うか守るかは固定じゃない。関係で決まる」

## ヨルダン近現代史

### 主要都市の歴史的役割
| 都市 | 歴史的役割 |
|------|-----------|
| アンマン | 都市の起源＋ローマ支配の象徴 |
| ペトラ（ワディムサ） | 古代交易国家（ナバテア）の頂点 |
| ワディラム | 砂漠景観＋近代史（アラブ反乱・ロレンス） |
| アカバ | 海上交易＋近代戦争（アカバの戦い） |

→ この4都市でヨルダンの歴史が一通りカバーできる

### アラブ反乱とロレンス（ワディラム）
- T.E.ロレンス（アラビアのロレンス）がアラブ反乱を支援した舞台
- アカバの戦い：中東近代史の転換点
- フウェイタート族（Huweitat）がロレンスとともにオスマン帝国に抵抗・アカバ攻略に協力 — 部族の誇りとして今も語り継がれる

### ベドウィン定住化の歴史
- 1950〜70年代：ヨルダン政府による定住化政策が本格化。国境整備で伝統的遊牧ルートが消滅
- ヨルダン軍（Arab Legion）：ベドウィン部族出身者を積極登用 → 定住・都市化の契機に
- 20世紀を通じて：国境・教育・医療・経済が定住を促進し、完全遊牧生活はほぼ消滅

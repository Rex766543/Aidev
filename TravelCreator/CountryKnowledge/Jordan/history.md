# Jordan — 歴史・遺跡・王朝

## ナバテア王国

### 一言で
砂漠を通るビジネスを全部握っていたプレイヤー。「砂漠版GAFA」的存在。

### なぜ歴史的に重要か
- 東西交易（香料・絹）を支配
- ローマに匹敵する経済力を持った
- 砂漠で都市を成立させた

### 何をしていたか（3レイヤー）
| レイヤー | 内容 | 現代の例え |
|---------|------|-----------|
| 運送 | キャラバンで商品を運ぶ | ヤマト運輸 |
| 売買 | 南アラビアで香料を安く買い、北（地中海側）で高く売る | 商社・転売 |
| インフラ支配 | 水供給・ルート独占・安全確保 | 高速道路会社＋警備 |

→ この3つを**全部**やっていたのがナバテアの強み

### なぜ砂漠で都市が成立したか
- 立地：紅海とアカバ方面、シリア・地中海、アラビア内陸を結ぶ交易結節点
- 水工学：周辺の泉（アイン・ムーサ等）＋雨季の洪水水を徹底回収・制御
  - diversion dam（洪水制御ダム）
  - Muthlim tunnel（トンネル）
  - 導水路（aqueducts）
  - 貯水槽・地下シスタン
- 戦略：遊牧的機動力を持ちながら、定点インフラを構築して交易を独占

### 盗賊との関係（重要）
略奪より「継続課金」の方が儲かる論理で、盗賊をエコシステムに組み込んだ。
- 通行料を支払う（関所的な非公式税）
- 部族を護衛として契約し「警備会社化」
- ルート・水を握ることで他勢力を従属させる

→ 盗賊 vs 商人ではなく、**同じエコシステム内の役割分担**

### 文化的特徴（重要3つ）
1. **水管理**：雨水の貯水・導水 → 砂漠で都市を成立
2. **岩削り建築**：建築ではなく"削る"（ペトラ）
3. **多文化融合**：ギリシャ＋中東＋ローマ → 国際交易都市として顧客適応

### なぜ多文化を受け入れたか
→ **ビジネスのため**（戦略的合理主義）
- 交易の中心 = 文化の交差点
- 来る商人に合わせた「分かりやすい文化」への適応
- 実利重視・排他的でない宗教観
- 現代でいう「グローバル企業のローカライズ」に相当

### なぜ世界史の教科書に出ないか
- 大帝国・制度・普遍史を主軸にする叙述では主役化しにくい
- フェニキア人のほうがアルファベット・地中海史と結びつけやすく扱いやすい
- 106年にローマに併合され「属州アラビア」として歴史に吸収されて見えにくくなった
- 重要性の問題ではなく、物語の作り方の問題

### ローマへの併合（106年）
- ローマ帝国に属州アラビアとして組み込まれる
- その後はローマ史・ビザンツ史に吸収

## ペトラ（ナバテア王国の首都）

### 位置づけ
世界遺産。中東最重要遺跡の一つ。「ただの遺跡」ではなく「水制御で成立した商業インフラ都市」として見ると理解が深まる。

### 主要スポット
| スポット | 意義 |
|---------|------|
| エル・ハズネ（宝物殿） | ペトラの象徴。岩を彫って作った高度技術 |
| アド・デイル（修道院） | 最大級建造物。交易都市のスケール感が分かる |
| ペトラ遺跡全体 | ナバテア王国の首都 |

## ベドウィンと砂漠社会

### ベドウィンとは
「民族名」ではなく**生活スタイル（遊牧・砂漠適応）の属性名**。アラビア語の「Badū」に由来。
- ラクダで移動
- 水源知識を持つ
- 部族単位で動く
- 名誉・客人保護・部族連帯が重要規範

### 現在のヨルダンでのベドウィン
- 完全遊牧の人は少なく、多くは定住化
- 人口の1割未満（Britannica）
- 国家形成・軍の文脈で重要な位置を占めてきた

### ナバテアとベドウィンのつながり
ナバテア人も初期には遊牧・半遊牧的背景を持つとされる。
→「移動する砂漠民が、交易の必要から定点インフラを持ち始めた」という連続性がある

### 客人保護（ホスピタリティ文化）とは
砂漠社会の根本的な生存ルール。
- 来た人を守り、もてなす義務（敵でも数日間は無条件保護）
- 食事・水を提供する
- 背景：砂漠は水・食料がなく、単独では死ぬ → 相互扶助が生存条件
- 機能：信用インフラ（ホテル＋警備の役割）
- 中東・砂漠文化全体に共通するルール（ベドウィンに特に強い）

### ベドウィンと聖カトリーナ修道院（シナイ）
ベドウィンが修道院を襲撃した理由：
1. 生存（食料・水・物資確保）
2. 砂漠の孤立した「資源拠点」として価値がある
3. 国家統治が弱い空白地帯の論理（力ある側が取る）
4. 襲撃 → 保護契約への交渉手段

実態：多くの時代で、ベドウィンは修道院を**保護**もしていた（食料・ガイド・警備提供）。
→ **敵でもあり、パートナーでもある**関係

砂漠社会の本質：「奪うか守るかは固定じゃない。関係で決まる」

## アンマン — 積層都市の歴史
*Related: Amman*

### 一言で
「ただの中東の首都」に見えるが、実は複数の文明が次々と上書きした「積層都市」。ペトラのような単一巨大遺跡ではなく、支配者が変わるたびに更新されてきた都市として見ると理解しやすい。

### 重なった文明の層
| 時代 | 支配・文化 | 遺構 |
|------|-----------|------|
| 古代（紀元前） | アンモン人の首都「ラバト・アンモン」 | シタデル起源 |
| ヘレニズム期 | プトレマイオス朝が「フィラデルフィア」と改名 | ギリシア的都市整備 |
| ローマ期（1〜3世紀） | ローマ帝国属州の地方都市として整備 | ローマ劇場・神殿 |
| ビザンツ期 | キリスト教的統治 | 教会跡 |
| ウマイヤ朝（7〜8世紀） | 初期イスラーム国家の宮殿建設 | シタデル内ウマイヤ朝宮殿 |
| 近代 | ヨルダン・ハシミテ王国の首都 | 現代都市 |

## フィラデルフィア — アンマンの古名とその語源
*Related: Amman*

### 語源
ギリシア語：**philo（愛する）+ adelphos（兄弟）= 「兄弟愛の都市」**

### アンマンがフィラデルフィアになった理由
- 元々はアンモン人の首都「ラバト・アンモン」
- アレクサンドロス大王以後のヘレニズム時代、**プトレマイオス朝エジプト**の支配下に入る
- 当時の王 **Ptolemy II Philadelphus**（称号の意味：「兄弟愛の人」）にちなんで「Philadelphia」と改名

### アメリカのフィラデルフィアとの関係
| 都市 | 命名由来 |
|------|---------|
| 古代アンマン（Philadelphia） | プトレマイオス2世の称号から |
| アメリカのPhiladelphia | 建設者William Pennが「宗教対立を超えた友愛・寛容」を理想としてギリシア語を採用 |

→ 同じギリシア語が語源。片方は王の称号、片方は理想思想という違い

## アンマン・シタデル（城塞）
*Related: Amman*

### 本質
「柱と廃墟があるだけ」ではなく、都市を見下ろす丘の上で支配者が変わるたびに宗教・行政・軍事機能が上書きされてきた「ヨルダン地域の支配史が積層した場所」。

### 上書きの歴史
- ローマ期 → ヘラクレス神殿（Jupiter系）建設
- ビザンツ期 → 教会建設
- ウマイヤ朝（8世紀） → 宮殿（Umayyad Palace）建設

### ウマイヤ朝宮殿とローマ要素
- 初期イスラーム国家はローマ・ビザンツの建築技術を大量継承したため、この宮殿にもローマ・ビザンツ的な要素（アーチ・ドーム構造）が色濃く残る
- 「イスラーム建築なのにローマっぽい」→ 初期イスラームの建築史の特徴そのもの

## ローマ劇場（アンマン）
*Related: Amman*

### 背景
- アンマンはローマ時代、デカポリス（10都市同盟）の一つ「フィラデルフィア」として整備された
- ローマ帝国は中東まで支配を広げ、属州各地にローマ式都市インフラを整備
- 劇場は娯楽施設でもあり、**「ローマ都市文明の象徴」** でもある
- 収容人数約6000人。今も野外イベントに使われる「生きた遺跡」

## ペトラ — 建築・地形詳細
*Related: Petra*

### Treasury（エル・ハズネ）の建設規模
- 高さ約40m、岩盤を直接彫り込んで作られた建造物
- 水利・石工・岩盤加工・都市設計など専門分化が必要な国家事業レベルの建設
- 王族・エリート層が主導したと考えられている
- 「宝物殿」という名前は後世の伝説から。実際は墓または神殿的施設

### シーク（峡谷入口）
- 幅1〜3m、高さ80m以上の断崖が続く峡谷
- 自然断層・侵食谷が基本（人工ではない）
- 水路・道路としてナバテア人が整備
- 風が強く感じるのは、谷地形による自然な気流圧縮効果

### 水路システム
- 微妙な傾斜を使った重力流システム
- 岩削り水路＋陶製パイプ（テラコッタ管）＋貯水槽＋洪水制御ダムの組み合わせ
- シークを流れる洪水の水を制御・貯水する高度な水利文明の産物

### 岩の黒い線
- 一見すると人工装飾・落書きのように見えるが、ほとんどが**自然地質**
- 鉄・マンガン等の鉱物が地層の差異として縞状に現れたもの（自然の地層・水流痕）

## ヨルダン近現代史

### 主要都市の歴史的役割
| 都市 | 歴史的役割 |
|------|-----------|
| アンマン | 都市の起源＋ローマ支配の象徴 |
| ペトラ（ワディムサ） | 古代交易国家（ナバテア）の頂点 |
| ワディラム | 砂漠景観＋近代史（アラブ反乱・ロレンス） |
| アカバ | 海上交易＋近代戦争（アカバの戦い） |

→ この4都市でヨルダンの歴史が一通りカバーできる

### アラブ反乱とロレンス（ワディラム）
- T.E.ロレンス（アラビアのロレンス）がアラブ反乱を支援した舞台
- アカバの戦い：中東近代史の転換点
- フウェイタート族（Huweitat）がロレンスとともにオスマン帝国に抵抗・アカバ攻略に協力 — 部族の誇りとして今も語り継がれる

### ベドウィン定住化の歴史
- 1950〜70年代：ヨルダン政府による定住化政策が本格化。国境整備で伝統的遊牧ルートが消滅
- ヨルダン軍（Arab Legion）：ベドウィン部族出身者を積極登用 → 定住・都市化の契機に
- 20世紀を通じて：国境・教育・医療・経済が定住を促進し、完全遊牧生活はほぼ消滅

# Jordan — 宗教・イスラム文化

## ヨルダンの宗教構成

### 国教
イスラム（スンニ派が中心）

### 少数派
- キリスト教徒が存在
- 憲法上、礼拝の自由は一定程度保障

### 宗教と社会の関係
- 国教 = 国家の制度的基盤
- 実際の社会の拠り所：宗教だけでなく、家族・親族・共同体・王制への安定期待も大きい
- 宗教倫理が日常規範に強く影響（礼儀・もてなし・対立回避）

## イスラーム商人の特徴（歴史的）

### 本質
宗教ベースの信用経済

### 強さの理由
- ムハンマド自身が商人出身 → 商業を肯定する文化的土台
- 広大な交易ネットワーク（インド〜東南アジア）
- 同じ宗教 = 信用担保（見知らぬ相手でも同じ礼拝・規範）
- 利子禁止 → 共同事業（リスク分散）
- ザカート（義務的寄付）→ 社会安定・共同体維持

→ 「信用のインフラ」を宗教で作った

## 客人保護とイスラム文化の関係

客人保護（ホスピタリティ）はイスラム以前からの砂漠文化のルールだが、
イスラムの倫理体系とも深く結びついている。
- 来訪者をもてなす = 神への奉仕の一形態
- 社会的信用・名誉とも直結

→ culture.md の「客人保護」も参照

## ナバテア人の宗教観

- 多神教（アラビア系の神々 + ギリシャ・ローマの神も受容）
- 実利重視・排他的でない → 交易相手の宗教文化にも柔軟に対応
- ペトラの建築：ギリシャ柱（コリント式）+ ローマ風ファサード + 中東的岩削り = 完全なミックス
- これは「信仰の混乱」ではなく「ビジネス戦略としての文化適応」

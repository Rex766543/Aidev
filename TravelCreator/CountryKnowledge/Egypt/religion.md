## 古代エジプトの来世思想とミイラ
*Related: Cairo (Giza)*

### 「なぜ身体が必要か」— 来世観の基本構造
古代エジプトの来世は「幽霊になる」のではなく、**身体を持ったまま別の世界で再生する**という物質的世界観。

| 構成要素 | 意味 |
|---------|------|
| **バ（Ba）** | 個性・人格。鳥の姿で表現される。死後も動き回れる |
| **カ（Ka）** | 生命力・活力。死後も維持される本人の「分身」 |
| **身体** | バとカが戻ってくる「器」。失われると存在が消滅する |

→ ミイラは「魂が戻ってくる器を保存するため」

### ミイラ製作の工程
1. 内臓を取り出し、カノポス壺（4つ）に別々に保管
2. 乾燥剤「ナトロン（天然炭酸ナトリウム）」に40日間漬けて乾燥
3. 樹脂・香料で処理し、リネン布で巻く
4. 副葬品を添えて棺に納める

## 古代エジプトの動物神
*Related: Cairo (Giza)*

### 「弱そうな動物がなぜ神か」への答え
古代エジプトの神々は「強さ」ではなく**「自然界の力・秩序・宇宙の仕組みの象徴」**として動物と結びついた。

| 動物 | 神名 | 象徴するもの |
|------|------|------------|
| 猫 | バステト | 家庭・守護・豊穣・母性 |
| ハヤブサ | ホルス | 空・太陽・王権（ファラオの守護神） |
| ワニ | ソベク | ナイルの力・水・軍事力 |
| スカラベ（コガネムシ） | ケペリ | 再生・太陽の運行（糞玉を転がす→太陽神が太陽を転がすイメージ） |
| ジャッカル | アヌビス | 死者・ミイラ化・冥界案内 |
| イビス（鳥） | トート | 知恵・文字・月 |

→ 動物そのものを崇拝しているのではなく、**その動物の性質を通して神の属性を表現**している

## 古代エジプトの復活思想（オシリス神話）
*Related: Cairo (Giza)*

### オシリス神話のあらすじ
1. オシリス（農業・死者の神）は弟セトに殺されてバラバラにされた
2. 妻イシスが遺体の断片を集め、アヌビスの助けで復活させた
3. オシリスは冥界の王として「死者の再生を司る神」になった

### これが何を意味するか
- 「死から復活できる」という実例の神話的提示
- 死者はオシリスと同じ旅をすれば再生できる → ミイラ・副葬品・呪文（死者の書）はすべてそのための準備

### 「来世ゾンビ化」ではない
- 来世は現世の延長ではなく「完全に再生した別の世界での生」
- 「オシリスの審判」：心臓をマアト（真理・正義の羽根）と天秤にかける
  - 心臓 < 羽根 → 楽園（アアル）へ
  - 心臓 > 羽根 → 怪物アメミト（ライオン・ワニ・カバの合体）に食べられて存在消滅

## シナイ山（ジャバル・ムーサ）— 三宗教の聖地
*Related: Mount Sinai*

### モーセとシナイ山
- 聖書（出エジプト記）：エジプトから脱出したイスラエル人を率いたモーセがここで神（YHWH）から**十戒**を授かった
- 「燃える柴」でモーセに神が現れた場所も山の近くとされる（聖カトリーナ修道院の「燃える柴」の場所）

### 三宗教での位置づけ
| 宗教 | モーセ（ムーサー）の位置づけ |
|------|--------------------------|
| ユダヤ教 | 律法（トーラー）を受け取った最重要預言者 |
| キリスト教 | 旧約聖書を継承。十戒はキリスト教倫理の土台 |
| イスラーム | コーランに最多登場する預言者（25回以上）。神と直接対話した「ムーサー」として超重要 |

→ 「イスラームとユダヤ教は別物」と思いがちだが、旧約系預言者を全員継承するのがイスラームの基本構造

## 聖カトリーナ修道院 — イスラームとキリスト教の共存
*Related: Mount Sinai*

### 概要
- 6世紀（565年頃）、ビザンツ帝国（ユスティニアヌス帝）の命で建設
- 世界最古の継続稼働修道院の一つ
- シナイ半島の山中に位置し、ギリシア正教の修道院として現在も機能

### イスラーム征服後の保護
- 7世紀、イスラームによるエジプト征服後も修道院は保護された
- モハメッドが修道院に保護を与えたとされる書状（アフドナーメ）が伝わる
- 理由：イスラームは「啓典の民」（キリスト教・ユダヤ教）を保護する義務がある（ジズヤ税を払えば信仰を認める）

### 修道院・ベドウィン・支配者の共生
- ベドウィンが修道院を時に襲撃、時に護衛した → 「敵でもありパートナーでもある」関係
- 修道院は食料・水を供給、ベドウィンは護衛・ガイドを提供
- 1,400年以上の共存の結果として現在に至る

## イスラム法と金（ゴールド）
- ハディースによる男性への金装飾の禁止：スンニ派で広く遵守
- 例外：男性の結婚指輪（銀・プラチナ推奨）、医療目的（歯の詰め物など）
- 女性には制限なし——これが文化・市場に与える影響は大きい（→ culture.md 参照）

# CLAUDE.md — TravelCreator

## このプロジェクトについて

エジプト・ヨルダン旅行中に感じた疑問や気づきを記録・深掘りするための個人用知識管理システム。
旅先でスマホのClaude Codeアプリからその場でメモを投げ込み、Claudeが自動で考察・整理してファイルに出力する。

### フォルダ構成

- `CountryKnowledge/` — 国・カテゴリ別の知識ベース（Egypt / Jordan）
- `Thought/` — Claudeが出力する考察ファイル置き場（YYYY-MM-DD_topic.md）

---

## トリガーとアクション

**「気付き」で始まる入力** → `.claude/skills/kizuki.md` を読み込んで処理する

**「CountryKnowledgeに追加して」「整理して」などの入力** →
1. 入力を国・カテゴリごとに分類する
2. 該当ファイルを読み込む（存在する場合）
3. 重複を除いて追記・整理する
4. 完了ファイル一覧を報告する

それ以外は通常通り応答する。
